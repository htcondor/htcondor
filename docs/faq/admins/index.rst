Which platforms does HTCondor support?
--------------------------------------

Supported platforms are listed in the :doc:`/overview/availability` section.

Is HTCondor cross-platform?
---------------------------

Yes.  Any supported version of HTCondor on any supported platform works with
any supported version of HTCondor on any supported platform.  However, not
all platforms have the same features; see the :doc:`/platform-specific/index`
for details.
